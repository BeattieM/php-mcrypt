Description
===========
The Chef PHP-Mcrypt cookbook installs and configures the php5_mcrypt module fro PHP.

Platform
========
* Debian, Ubuntu

Cookbooks
=========
* PHP

Usage
=====
To install Mcrypt add the php-mcrypt recipe to your run list.
